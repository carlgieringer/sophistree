# tapestry-highlights

```sh
# Run all tests
npm run test
# Debug tests (add `debugger` to client code and open devtools when the browser opens)
npm run test-debug
# Update snapshots
npm run test-update-snapshots
# Run specific tests
npm run test -- e2e/HighlightManager.spec.ts -g "should create a highlight" --project=chromium
# Run specific browsers
npm run test -- --project=chromium
```
